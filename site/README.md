# NameRank explainer site

An interactive React companion site for the NameRank paper, aimed at first-time
readers. Three sections:

1. **How it works** — jargon-free walkthrough of the mechanism (probe → judge →
   coverage×accuracy → panel average), illustrated with four real archetype
   responses (full recognition / partial / hallucination / refusal).
2. **Findings** — the four headline results as interactive charts (credential
   treadmill, artifact-over-creator inversion, h-index vs citations, corpus
   gradient) plus the 37-model leaderboard.
3. **Explorer** — all 5,719 entities with per-model score panels and gold
   answers; 615 verbatim judged responses with cross-judge verdicts; the full
   54-cohort × 37-model heatmap.

## Develop

```bash
npm install
npm run data    # regenerate JSON assets from ../data and ../experiments
npm run dev
```

## Build & deploy

```bash
npm run build   # outputs dist/ (base './', works at any mount path)
rsync -az --delete dist/ icourse:/var/www/ring0.me/research/namerank/
```

Live at https://01.me/research/namerank/

Data assets are generated by `scripts/build_data.py` (run from the repo root or
via `npm run data`): small chart data is imported at build time from
`src/data/`, large assets (entities, score matrix, judged cases, gold-answer
shards) are lazy-fetched at runtime from `public/data/`.

All numbers, responses, and judge rationales shown on the site are real records
from the study; the inversion table mirrors the paper's Table `tab:inversion`
verbatim.
